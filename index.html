Configuration Firebase
Suivez ces 4 étapes pour connecter HoopStat à votre base de données Firebase. À faire une seule fois.
Étape 1
Rendez-vous sur console.firebase.google.com et créez un nouveau projet (désactivez Google Analytics, inutile ici).
Étape 2
Dans votre projet : Build → Realtime Database → Create database. Choisissez la région la plus proche, puis sélectionnez Start in test mode.
Étape 3
Allez dans Project Settings ⚙ → Your apps → Add app → Web </>. Donnez un nom, puis copiez les valeurs affichées dans le bloc firebaseConfig.
Étape 4
Collez chaque valeur ci-dessous, puis cliquez sur Enregistrer et lancer.
